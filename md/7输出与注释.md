### 输出与注释

```python
print('he\
aaa\
aaa')

# 这是一个打印语句，请你看见了不要慌张
# 这是一个注释
# 注释会被解释器所忽略
# print(123+456) 这行代码被注释了，将不会执行
print('abc') # 这是一个单行注释
```

### 结果

``` python
heaaaaaa
abc
```
