### 求平方根

```python
#计算负数或者复数
import cmath        # 导入复数数学模块
num=int(input("请输入一个数："))
num_sqrt=cmath.sqrt(num)
print('{0}的平方根为{1:0.3f}+{2:0.3f}j'.format(num,num_sqrt.real,num_sqrt.imag))
```

### 结果

``` python
请输入一个数：4
4的平方根为2.000+0.000j
```
