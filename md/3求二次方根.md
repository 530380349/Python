### 求二次方根

```python
# 导入 cmath(复杂数学运算) 模块 
import cmath 
a = float(input('输入 a: ')) 
b = float(input('输入 b: ')) 
c = float(input('输入 c: ')) 
# 计算 
d = (b**2) - (4*a*c) 
# 两种求解方式 
sol1 = (-b-cmath.sqrt(d))/(2*a) 
sol2 = (-b+cmath.sqrt(d))/(2*a) 
print('结果为 {0} 和 {1}'.format(sol1,sol2))
```

### 结果

``` python
输入 a: 2
输入 b: 8
输入 c: 3
结果为 (-3.58113883008419+0j) 和 (-0.41886116991581024+0j)
```
