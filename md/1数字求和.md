### 数字求和

```python
num1=input("输入第一个数字：")
num2=input("输入第二个数字：")

#求和
sum=float(num1)+float(num2)

#显示结果
print("数字{0}和{1}相加结果为：{2}".format(num1,num2,sum))
```

### 结果

``` python
输入第一个数字：3
输入第二个数字：4
数字3和4相加结果为：7.0
```
