### 判断是否为闰年

```python
# 整百年能被400整除的是闰年
# 非整百年能被4整除的为闰年
year = int(input("输入一个年份: "))
if (year % 4) == 0:
    if (year % 100) == 0:
        if (year % 400) == 0:
            print("{0} 是闰年".format(year))
        else:
            print("{0} 不是闰年".format(year))
    else:
        print("{0} 是闰年".format(year))
else:
    print("{0} 不是闰年".format(year))
```

``` python
输入一个年份: 2018
```
