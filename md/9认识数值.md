### 认识数值

```python
a = 10
b = 20
# 如果数字的长度过大，可以使用下划线作为分隔符
c = 123_456_789
print(a,b,c)
c = 0b10 # 二进制的10
print(c)
# 八进制 0o开头
c = 0o10
print(c)
# 十六进制 0x开头
c = 0x10
print(c)
c = -100
c = c + 3
print(c)
c = 1.23
print(c)
c = 0.1 + 0.2 # 0.30000000000000004
print(c)
```

### 结果

``` python
10 20 123456789
2
8
16
-97
1.23
0.30000000000000004
```
